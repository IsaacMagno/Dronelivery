# Dronelivery

#### Dronelivery é um aplicativo de entregas com drones, que desenvolvi durante a aceleração Java na Trybe

#### Principais Tecnologias utlizadas no desenvolvimento
- React
- Redux
- Tailwind CSS
- Java
- MySQL
- Quarkus
